# EnemySpawnConfig

Configure enemy spawn amounts. Global and per-difficulty multipliers, plus peaceful mode.

R.E.P.O. BepInEx mod. Host-side only.

## Install

Requires [BepInEx 5.x](https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/).

Place `EnemySpawnConfig.dll` in `BepInEx/plugins/`.

## Configuration

Edit `BepInEx/config/maxenterme.EnemySpawnConfig.cfg` after first launch.

## Build

```
dotnet build -c Release
```
